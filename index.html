01.About Me

Hi, I'm Rushabh, a DevOps Engineer based in Ahmedabad, India, currently at Oro, building reliable cloud infrastructure on Kubernetes with Helm, Argo CD, Terraform, and GitHub Actions. I'm a Docker Captain (selected 2026, ~220 members worldwide) and a Grafana Champion (inaugural 2025 cohort, ~110 worldwide).

My focus areas are Kubernetes, CI/CD, IaC, observability with the Grafana LGTM stack (Loki, Grafana, Tempo, Mimir) and OpenTelemetry, and cloud cost optimization. Before Oro, I drove a 59% AWS spend reduction at Genuin, led a zero-downtime AWS→OCI migration under a hard deadline, and built ISO / SOC Type 1 readiness controls. I started out as a Laravel/PHP developer, then moved into DevOps via Tridhya Tech and KodeKloud, where I represented the team at KubeCon India 2024. I organize Grafana & Friends Ahmedabad, run workshops on Prometheus and the LGTM stack, write a bi-weekly newsletter, and contribute to open source, including translating eBPF.io into Hindi.

$ now > deep in eBPF, OpenTelemetry, and tuning Grafana LGTM stacks for production reliability.
